Prompt Tips

  • Be specific: "a tabby cat sitting on a windowsill, morning light, soft focus" beats "a cat"
  • Style keywords: Add "photorealistic", "digital painting", "watercolor", "cinematic lighting" etc.
  • Negative prompt: Use to remove common problems — "blurry, low quality, text, watermark, deformed hands"
  • Quality boosters: "highly detailed, 8k, sharp focus, professional photography"

CFG Scale Guide

  • 1-4: Creative, loose interpretation of prompt
  • 5-7: Balanced — follows prompt while staying natural (recommended)
  • 8-12: Strict prompt following, can look over-processed
  • 13+: Usually too high — saturated colours, artifacts

Model Strengths

  • SD 1.5: Fast, lightweight. Good for stylised art and quick iterations
  • SDXL Base: All-rounder. Better detail and composition than SD 1.5
  • Juggernaut XL: Photorealism specialist. Skin, materials, lighting
  • RealVisXL: Portrait and scene realism. Lower CFG (4.5-6) for natural look
  • Nano Banana: Best prompt understanding. Handles complex scenes well

Reference Images

Upload 1-4 reference images to guide the style/composition. Works with ComfyUI (IP-Adapter) and Gemini (native multimodal). Set IP-Adapter strength 0.3-0.6 for subtle influence, 0.7+ for strong style matching.
