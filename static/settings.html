Setup guide:
1. Install Ollama: curl -fsSL https://ollama.com/install.sh | sh
2. Pull a model: ollama pull qwen2.5:14b
3. Ollama runs automatically as a service. Enter the URL above.

Verified models: qwen2.5:14b (best quality), qwen2.5:7b (faster), mistral:7b, llama3:8b
Any Ollama-compatible model works — experiment with what you have installed.
